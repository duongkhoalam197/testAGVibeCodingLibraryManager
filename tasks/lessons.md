# Self-Improvement Lessons

## Date: 2026-03-30

### Pattern: Always Read the Full Context
**Problem**: The model provided a PR review and implemented features before fully reading the instructions provided in the `.antigravity/` directory.
**Context**: This project has specific automated command rules and documentation standards (e.g., `CONTEXT.md` for features, specific header for `PROJECT-STATUS.md`).
**Lesson**: BEFORE performing ANY action or starting a new session, read ALL files in the `.antigravity/` and `docs/` folders to ensure absolute compliance with project protocols. Never assume a continuous conversation doesn't require checking the "Startup" or "Command" rules if they exist.
**Actionable Rule**: At the start of any project interaction, list and read the contents of any configuration/command directory (like `.antigravity/`, `.ai/`, or `.cursor/`) immediately.
